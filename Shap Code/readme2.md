SHAP Feature Importance Analysis

Overview

This repository analyzes feature importance using SHAP for various classifiers like CatBoost, XGBoost, LightGBM, SVM, and more. It calculates and visualizes feature importance across classifiers, focusing on channel-wise importance and aggregated importance.

Files
	•	Channel Wise Features Importance across Classifiers utilizing SHAP.py
	•	Channel Wise Importance across Classifiers utilizing SHAP.py
	•	Features Importance across Classifiers utilizing SHAP.py

SHAP Analysis Process
	•	Top 10 Features: Displays top features by SHAP importance.
	•	Channel-wise Importance: Aggregates SHAP values by EEG channel lead.
	•	Aggregated Feature Importance: Sums SHAP importance across all leads for each feature.
